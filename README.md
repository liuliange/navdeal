# NavDeal · 特惠好物导航

> 基于 Notion 数据库的优惠导航网站 —— 发现好价，一站直达。

## ✨ 特性

- 📦 使用 Notion 作为数据库，无需单独部署后端
- 🎨 简约现代界面，专注优惠信息展示
- 📱 响应式设计，桌面/移动端完美适配
- 🌙 深色/浅色主题切换
- 🚀 一键部署到 Vercel

## 🛠️ 技术栈

- Next.js
- TypeScript
- Tailwind CSS
- Notion API

## 🏃 本地开发

bash

pnpm install

pnpm dev

访问 http://localhost:3000 即可预览。

## 📦 构建部署

bash

pnpm build
